    Class AuthorizationCodeCredential

    Authenticates by redeeming and authorization code previously obtained from Azure Acitve Directory. See https://docs.microsoft.com/en-us/azure/active-directory/develop/v2-oauth2-auth-code-flow for more information about the autorization code authentication flow.
