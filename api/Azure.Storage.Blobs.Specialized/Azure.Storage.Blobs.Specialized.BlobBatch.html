    DeleteBlob(Uri, Nullable<DeleteSnapshotsOption>, Nullable<BlobAccessConditions>)

    The DeleteBlob(Uri, Nullable<DeleteSnapshotsOption>, Nullable<BlobAccessConditions>) operation marks the specified blob or snapshot for deletion. The blob is later deleted during garbage collection.

    Note that in order to delete a blob, you must delete all of its snapshots. You can delete both at the same time using Include.

    SetBlobAccessTier(Uri, AccessTier, Nullable<RehydratePriority>, Nullable<LeaseAccessConditions>)

    The SetBlobAccessTier(Uri, AccessTier, Nullable<RehydratePriority>, Nullable<LeaseAccessConditions>) operation sets the tier on a blob. The operation is allowed on block blobs in a blob storage or general purpose v2 account.

    A block blob's tier determines Hot/Cool/Archive storage type. This operation does not update the blob's ETag. For detailed information about block blob level tiering .
